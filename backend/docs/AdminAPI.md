# LocalLens AI / Terra Vision — Admin & Operations API Specification

Lead Engineer: Pinank Shah (`pinankshah-Ab`)

---

## Endpoints Summary
- `GET /api/v1/admin/metrics`: Fetch centralized operational dashboard metrics.
- `GET /api/v1/admin/users`: List all platform users with RBAC roles and status.
- `GET /api/v1/admin/reports`: List pending content moderation reports.
- `GET /api/v1/admin/ai-ops`: Fetch AI provider token usage, latency, and cost estimates.
- `GET /api/v1/admin/audit-logs`: Fetch security audit trail logs.
