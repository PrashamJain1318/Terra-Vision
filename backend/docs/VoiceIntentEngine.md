## Intent Classification Pipeline
1. Acoustic Speech Perception / STT Audio Buffer.
2. Natural Language Tokenization & Entity Extraction.
3. Module Intent Classification (`PLANNER_CREATE`, `FOOD_DISCOVER`, `SAFETY_CHECK`, `VISION_TRANSLATE`, `MAPS_NAVIGATE`, `MEMORY_SAVE`).
4. Microservice Payload Execution & Context Synchronization.
