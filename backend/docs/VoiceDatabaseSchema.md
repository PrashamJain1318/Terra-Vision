Developer: Pinank Shah (`pinankshah-Ab`)

## MongoDB Schemas
- `VoiceSession`: Active audio interaction session.
- `VoiceMessage`: Audio transcript and intent message log.
- `VoiceIntent`: Registered natural language intent schema.
- `VoiceTranslation`: Speech translation cache.
- `VoiceContext`: Trip & module context memory state.
