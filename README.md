## 🌟 Key Product Features

### 1. Google Maps AI Discovery Engine V2 (`/dashboard/maps`)
- **Command Search Bar (`⌘ K`)**: Google Autocomplete -> Geocoding -> Google Places -> Gemini AI enrichment pipeline without page refresh. Voice Search mic integration included.
- **Hero Map Canvas (75% Width)**: 24px rounded dark glass frame with live Satellite, Terrain, Traffic, Zoom, Compass, User Location, and Search-this-Area controls.
- **Custom Animated Map Markers**: Categories for 🍽 Restaurants, 🛕 Temples/Historical, 🌿 Nature, 🏛 Museums, 🏖️ Beaches, 🏨 Hotels, and 🧗 Adventure.
- **Right Apple-Style AI Sheet**: Hero photo gallery, Google Reviews count, and **AI Intelligence Matrix Cards** (*Photography 9.6/10, Crowd Low, Budget ₹₹, Adventure 9.1, Family Yes, Safety 9.8*).
- **Floating Bottom Route Planner**: Multi-modal drawer (*Driving 🚗, Walking 🚶, Cycling 🚴, Transit 🚌*) with ETA, distance, and fuel cost estimation.

### 2. Multi-Agent AI System (`/dashboard/os`)
Orchestrates 8 specialized AI agents:
- `PlannerAgent`: Itinerary generation & multi-day scheduling.
- `DiscoveryAgent`: Spatial discovery & hidden gem extraction.
- `SafetyAgent`: Geofence risk monitoring & emergency helplines (112, 108, 1364).
- `FoodAgent`: Authentic culinary matching & dietary filtering.
- `MemoryAgent`: Memory timeline creation & journey replay.
- `CommunityAgent`: Verified traveler reviews & group trip matching.
- `EnterpriseAgent`: Partner hotel/restaurant booking gateway.
- `PersonalizationAgent`: HSL preference DNA & custom recommendations.

### 3. Enterprise & Partner Platform (`/dashboard/enterprise`)
Multi-tenant portal for hotel partners, restaurant managers, tourism boards, and local guides featuring RBAC extension, campaign management, and API gateway access.

### 4. Smart Travel Companion Ecosystem (`/dashboard/ecosystem`)
Offline-first synchronization engine supporting Android Auto, Apple CarPlay, Smartwatch readiness, travel widgets, background location awareness, and emergency offline mode.

## ⚙️ Quick Start Installation

```bash
# 1. Clone the repository
git clone https://github.com/PrashamJain1318/Terra-Vision.git
cd Terra-Vision

# 2. Setup Backend Server
cd backend
npm install
PORT=5050 npm run start

# 3. Setup Frontend Web App (In a new terminal window)
cd ../frontend
npm install
npm run dev
```

Visit `http://localhost:3000/dashboard/maps` in your browser!
